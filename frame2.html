This WebApplication is a simple JMS Test Tool.

1. Type in the JNDI name of the Queue Connection Factory resource reference that has been assigned to the application during deployment or modifed in the application resource reference.

2. Type in the JNDI name of the Queue DEstination resource reference that has been assigned to the application during deployment or modifed in the application resource reference.

3. Click 'Put', to send a message (act as a JMS Producer)

4. Click 'Get', to receive message(s) on the queue (act as a JMS Consumer)
